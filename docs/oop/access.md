# سطوح دسترسی

| سطح دسترسی           | دسترسی از کلاس خودش | کلاس‌های فرزند | بیرون از کلاس  |
| -------------------- | ------------------- | -------------- | -------------- |
| `private`            | ✅                   | ❌              | ❌              |
| `protected`          | ✅                   | ✅              | ❌              |
| `public`             | ✅                   | ✅              | ✅              |
| `internal`           | ✅                   | ✅ (درون پروژه) | ✅ (درون پروژه) |
| `protected internal` | ✅                   | ✅              | ✅ (درون پروژه) |
| `private protected`  | ✅                   | ✅ (درون پروژه) | ❌              |


:::note
3 مورد آخر خیلی در یونیتی استفاده نمی شوند.
:::
