# رندر کردن خطوط (Rendering lines)

کامپوننت Line Renderer (رندرکننده خط) آرایه‌ای از دو یا چند نقطه در فضای سه‌بعدی را می‌گیرد و بین هر کدام یک خط مستقیم رسم می‌کند. شما می‌توانید از Line Renderer برای کشیدن هر چیزی، از یک خط صاف ساده گرفته تا یک مارپیچ پیچیده، استفاده کنید.

خط همیشه پیوسته است؛ اگر نیاز به کشیدن دو یا چند خط کاملاً مجزا دارید، باید از چندین GameObject (آبجکت بازی) استفاده کنید که هر کدام Line Renderer مخصوص به خود را داشته باشند.

Line Renderer خطوطی را که عرضشان بر حسب پیکسل باشد، رندر نمی‌کند. بلکه چندضلعی‌هایی (پلی‌گون‌هایی) را رندر می‌کند که عرضشان بر حسب واحدهای دنیای بازی (world units) است. Line Renderer از همان الگوریتمی برای رندر کردن خطوط استفاده می‌کند که Trail Renderer (رندرکننده ردپا) استفاده می‌کند.

:::tip
هر گیم آبجمت فقط می تواند مسئولیت کشیدن یک خط را به عهده داشته باشد.
البته منظور از یک خط یک خط صاف نیست بلکه یک خط پیوسته است.
:::
:::tip
واحد های عرض این خصوص بر حسب پیکسل نیستند.
این به این معنی است که در همه‌ی دستگاه ها یکسان نمایش داده نمی شوند و با تغییر مثلا سایز دوربین ضخامت آن ها نیز تغییر می کند.
:::
