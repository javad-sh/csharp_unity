# استفاده از ویرایشگر

مستقیم‌ترین راه برای استفاده از ویرایشگر، شناسایی دستی عناصر است. اگر روی تصویر کلیک کنید، یک ناحیه انتخاب مستطیلی با دستگیره‌هایی در گوشه‌های آن ظاهر می‌شود. شما می‌توانید این دستگیره‌ها یا لبه‌های SpriteRect (مستطیل اسپرایت) را بکشید تا اندازه آن را حول یک عنصر خاص تغییر دهید. پس از جداسازی یک عنصر، می‌توانید با کشیدن یک SpriteRect جدید در بخش دیگری از تصویر، عنصر دیگری را اضافه کنید. متوجه خواهید شد که وقتی یک SpriteRect را انتخاب می‌کنید، یک پنل در پایین سمت راست پنجره ظاهر می‌شود.

کنترل‌های موجود در این پنل به شما اجازه می‌دهند یک نام برای گرافیک اسپرایت انتخاب کنید و موقعیت و اندازه SpriteRect را با استفاده از مختصات آن تنظیم نمایید. عرض حاشیه (border) برای سمت چپ، بالا، راست و پایین را می‌توان بر حسب پیکسل مشخص کرد. این حاشیه‌ها هنگام استفاده از تکنیک 9-Slicing (برش نُه تکه) برای اسپرایت‌ها مفید هستند. همچنین تنظیماتی برای pivot (نقطه محوری) اسپرایت وجود دارد که یونیتی از آن به عنوان مبدأ مختصات و «نقطه لنگر» اصلی گرافیک استفاده می‌کند. شما می‌توانید از بین چندین موقعیت پیش‌فرض نسبت به SpriteRect (مثلاً مرکز، بالا سمت راست و غیره) یکی را انتخاب کرده یا از مختصات سفارشی استفاده کنید.

دکمه Trim که در کنار آیتم منوی Slice قرار دارد، اندازه SpriteRect را طوری تغییر می‌دهد که بر اساس شفافیت (transparency)، به طور دقیق دور لبه‌های گرافیک قرار بگیرد.

نکته: حاشیه‌ها (Borders) فقط برای سیستم UI (رابط کاربری) پشتیبانی می‌شوند و نه برای کامپوننت Sprite Renderer دو بعدی.
