# باز کردن ویرایشگر اسپرایت (Sprite Editor)
برای باز کردن ویرایشگر اسپرایت:

1. تصویر دو بعدی (2D) که می‌خواهید ویرایش کنید را از پنجره پروژه (Project window) انتخاب کنید.
نکته: شما نمی‌توانید یک اسپرایت را با انتخاب آن در نمای صحنه (Scene view) ویرایش کنید.
2. روی دکمه Sprite Editor در پنجره بازرسی واردات بافت (Texture Import Inspector) کلیک کنید.
یونیتی ویرایشگر اسپرایت را نمایش می‌دهد.

نکات:

* شما فقط زمانی می‌توانید دکمه Sprite Editor را ببینید که Texture Type (نوع بافت) تصویری که انتخاب کرده‌اید روی Sprite (2D and UI) تنظیم شده باشد.
* اگر تصویر شما شامل چندین عنصر است، Sprite Mode را در پنجره بازرسی واردات بافت روی Multiple تنظیم کنید.


همراه با تصویر ترکیبی، شما تعدادی کنترل در نوار بالای پنجره خواهید دید. اسلایدر (لغزنده) در بالا سمت راست، بزرگ‌نمایی (zoom) را کنترل می‌کند، در حالی که دکمه نوار رنگی در سمت چپ آن، انتخاب می‌کند که خود تصویر را ببینید یا سطوح آلفای (alpha levels) آن را. راست‌ترین اسلایدر، پیکسلی شدن (mipmap) بافت (Texture) را کنترل می‌کند. حرکت دادن این اسلایدر به سمت چپ، وضوح (resolution) بافت اسپرایت را کاهش می‌دهد. مهم‌ترین کنترل، منوی Slice در بالا سمت چپ است که به شما گزینه‌هایی برای جداسازی خودکار عناصر تصویر ارائه می‌دهد.

دکمه‌های Apply و Revert را به ترتیب برای ذخیره کردن یا نادیده گرفتن تغییراتی که در پنجره Sprite Editor ایجاد کرده‌اید، انتخاب کنید. شما می‌توانید این دکمه‌ها را طوری تنظیم کنید که قبل از اعمال اثراتشان، یک کادر محاوره‌ای تأیید نمایش دهند. برای این کار، در پنجره Preferences یونیتی، به مسیر 2D > Sprite Editor Window بروید تا گزینه‌های پنجره Sprite Editor را باز کنید.

گزینه مربوطه را انتخاب کنید تا پس از انتخاب Apply یا Revert، یک کادر محاوره‌ای برای تأیید انتخاب شما ظاهر شود و از ذخیره یا لغو تصادفی تغییرات ایجاد شده جلوگیری کند.
